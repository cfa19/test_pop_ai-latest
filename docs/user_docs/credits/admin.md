# Credit Administration

Organization Admins have access to a finance administration dashboard with detailed
statistics and transaction history.

---

## Access the admin dashboard

Navigate to **Dashboard > Finance & Analytics > Admin**.

This page is available to the **Org Admin** role.

---

## Statistics overview

The admin dashboard displays key metrics:

| Metric | Description |
|---|---|
| **Total transactions** | Number of all credit transactions |
| **Total credits purchased** | Sum of all purchased credits |
| **Total credits consumed** | Sum of all consumed credits |
| **Active packs** | Number of currently available credit packs |

---

## Transaction history

A table of the 20 most recent transactions within your organization, showing type, amount,
user, date, and reason.


---

## Active credit packs

A list of currently available credit packs with their names, credit amounts, and prices.
