# Analytics Overview

POPskills provides role-adaptive analytics dashboards that surface the metrics most relevant
to each user type.

---

## Access analytics

Navigate to **Dashboard > Finance & Analytics > Analytics**.

Analytics are available to the **Consultant** and **Org Admin** roles.

---

## Consultant view

Title: *"Analysis & Reporting — Consultant"*

### Statistics cards

| Metric | Description |
|---|---|
| **Total beneficiaries** | Number of assigned beneficiaries |
| **Total activities** | Activities created or managed |
| **Completed activities** | Activities finished by beneficiaries |
| **In-progress activities** | Activities currently being worked on |
| **Average score** | Mean score across all completions |
| **Total time spent** | Cumulative time spent by all beneficiaries |
| **Success rate** | Percentage of scores at or above 60% |

### Charts

- **Status distribution** -- Breakdown of activity statuses (completed, in progress, not started)
- **Trend charts** -- Completion and score trends over time
- **Results analytics** -- Detailed breakdown by category and activity type

Use the **period selector** to filter data by time range.

---

## Organization Admin view

Title: *"Analysis & Reporting — Organization"*

Shows the same components as the consultant view but scoped to the entire organization:

- All users, activities, and completions within the organization
- Organization-wide score averages and success rates
