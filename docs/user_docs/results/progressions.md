# Real-Time Progressions

The progressions page gives consultants and admins a live view of all activities currently
being worked on by beneficiaries.

---

## Access progressions

Navigate to **Dashboard > Management > Progressions**.

This page is available to the **Consultant** role.

---

## Statistics

Two summary cards appear at the top:

| Metric | Description |
|---|---|
| **Total in progress** | Number of activities currently being worked on |
| **Active today** | Number of activities with recent updates today |

---

## Progressions table

Each row in the table shows:

- **Beneficiary** name and email
- **Activity** title and type
- **Progress** bar with percentage
- **Last update** time (e.g., "Just now", "5 min ago", "2 hours ago", "1 day ago")

---

## View a live activity

Click any row to open the activity in **read-only consultant view**. You can see exactly
what the beneficiary is working on and how far they have progressed, without being able to
modify their work.

Use this page to identify beneficiaries who may be stuck or need guidance. If a progress bar has not moved in a while, it may be a good time to reach out.
