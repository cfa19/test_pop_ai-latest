# Consultant Results Dashboard

The results dashboard gives consultants and organization admins a centralized view of all
completed activities.

---

## Access the dashboard

Navigate to **Dashboard > Management > Results**.

---

## Statistics overview

Five summary cards appear at the top:

| Metric | Description |
|---|---|
| **Total** | All completed activity results |
| **Completed today** | Results submitted today |
| **Average score** | Mean score across all results |
| **Published** | Results that have been released to beneficiaries |
| **Pending** | Results awaiting review |

---

## Results table

A paginated table lists all completed activities. Each row shows:

- **Beneficiary** name and email
- **Activity** title and type
- **Score** (numerical)
- **Completion date**
- **Time spent**
- **Release status** -- Pending review, Released, or Scheduled
- **Urgency flag** -- High or low, based on intervention flags

You can filter results by release status.

---

## What you see depends on your role

| Role | Visible results |
|---|---|
| **Consultant** | Results from their assigned beneficiaries |
| **Org Admin** | Results from all users in their organization |
