# Create an Activity

Consultants and Organization Admins can create new activities and assign them to beneficiaries
or make them publicly available.

---

## Access the creation form

Navigate to **Dashboard > Activities** and click **Create activity**.

---

## Fill in activity details

1. Enter an activity **Title**.
2. Write a **Description** explaining what the activity covers and its objectives.
3. Select a **Category** from the dropdown (e.g., Evaluation, Accompaniment).
4. Choose an **Activity type** (e.g., CV Analyzer, VAE Livret 2).
5. Set the **Credit cost** — the number of credits consumed per completion.
6. Configure additional settings if the selected activity type requires them.
7. Click **Save**.

The available activity types depend on what is configured on the platform.

---

## Activity settings

| Setting | Description |
|---|---|
| **Title** | The name displayed to users |
| **Description** | Explanation of the activity's purpose |
| **Category** | Used for filtering in the catalog |
| **Activity type** | The type of exercise (e.g., CV Analyzer, VAE Livret 2) |
| **Credit cost** | Credits consumed per run |
| **Active** | Whether the activity appears in the catalog |
| **Featured** | Highlight the activity in the catalog |
| **Tags** | Keywords for search and organization |

---

## Edit an activity

1. Navigate to **Dashboard > Activities**.
2. Click the activity you want to modify.
3. Click **Edit**.
4. Make your changes and click **Save**.

Editing an activity does not affect completions that are already in progress or finished. Changes apply only to future runs.
