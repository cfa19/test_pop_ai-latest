# Activities Overview

Activities are the core building blocks of POPskills. Each activity is an interactive exercise
that guides you through a structured workflow.

---

## What is an activity?

An activity is a self-contained professional development exercise. Examples include:

- Analyzing a CV to extract structured information
- Building a VAE Livret 2 certification dossier
- Completing a RIASEC personality assessment
- Running a career-fit simulation
- Preparing for a professional interview

Each activity has a **type**, a **category**, and a **credit cost** that determines how many
credits are consumed when it is run.

---

## Activity lifecycle

```
Created --> Assigned --> Started --> Completed --> Validated --> Released
```

1. **Created** -- A consultant or admin creates the activity and configures it.
2. **Assigned** -- The activity is assigned to a beneficiary (or the user self-selects it).
3. **Started** -- The user launches the activity and begins the workflow.
4. **Completed** -- The user finishes all steps. Results are saved automatically.
5. **Validated** -- A consultant reviews, enriches, and validates the results.
6. **Released** -- The results are published to the beneficiary with recommendations.

Individual users who are not part of an organization skip the assignment and validation steps. Their results are available immediately after completion.

## Key concepts

| Term | Definition |
|---|---|
| **Credit cost** | Number of credits consumed when the activity is run |
| **Category** | Classification used for filtering (e.g., Evaluation, Accompaniment) |
| **Auto-save** | Progress is saved automatically so users can resume later |
