# Run an Activity

Running an activity launches a step-by-step guided workflow. Each activity type is different,
but they all share common behaviors described here.

---

## Launch an activity

1. Navigate to **Dashboard > Activities**.
2. Click the activity you want to start.
3. Click **Run** (or **Resume** if you have a saved session).

The activity opens in a full-screen immersive mode.

---

## Common features

### Auto-save

Your progress is saved automatically every few seconds. You can safely close the browser and
return later — your work will be restored.

A save indicator in the interface confirms when your data has been saved.

### Progress tracking

Most activities display a progress bar or step indicator so you know where you are in the workflow.

### Credit consumption

Credits are consumed as the activity processes your data (e.g., analysis, generation). The
activity card shows the credit cost before you start.

Make sure you have enough credits before starting an activity. If your balance is insufficient, you will see an error message.

---

## Exiting an activity

If you need to leave before finishing:

1. Click the **exit** button or navigate away.
2. A confirmation dialog appears: *"Are you sure you want to leave? Your progress will be saved."*
3. Confirm to exit. You can resume later from where you left off.

---

## Completing an activity

When you finish all steps, your results are saved and you are redirected to a **completion
page** with a congratulations message.

If the activity is part of a **program**, you are redirected back to the program page to
continue with the next activity.

If a consultant is assigned to your account, your results enter a **validation queue** before being released to you.

---

## Token access (no account required)

If you received a **token link** from a consultant, you can complete an activity without
creating an account:

1. Open the link shared with you.
2. You are taken directly to the activity — no login needed.
3. Complete the activity as usual (auto-save, progress tracking, etc.).
4. On completion, you see an end-of-activity page with a custom message.

Token access is limited to the specific activity linked. You will not have access to the dashboard, results page, or other platform features.

---

## Consultant view mode

Consultants can view a beneficiary's in-progress or completed activity in **read-only mode**.
A blue banner indicates: *"Consultation mode — You are viewing the beneficiary's activity in
read-only."*
