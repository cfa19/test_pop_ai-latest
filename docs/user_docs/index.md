# POPskills User Documentation

Welcome to the **POPskills** user documentation. This guide covers every feature of the
platform, organized by functional domain.

POPskills is a professional development platform that enables organizations, consultants,
and individuals to create, assign, and track interactive activities — from CV analysis to
certification preparation.

---

## Getting Started

Set up your account and take your first steps on the platform.

- Create an Account -- Sign up as an individual, consultant, or organization
- Log In -- Access your account
- Reset Your Password -- Recover access to your account
- First Steps -- Navigate the dashboard and discover key features

---

## Activities

Browse, create, and run professional development activities.

- Activities Overview -- Understand how activities work
- Browse Activities -- Find and filter available activities
- Create an Activity -- Set up a new activity (Consultant / Admin)
- Run an Activity -- Complete an activity step by step
- Activity Tokens -- Share activities via secure links (no account required)

## Programs

Organize activities into structured learning paths.

- Programs Overview -- What programs are and how they work

- Enroll in a Program -- Browse and join public programs
- Manage Assignments -- Assign activities to beneficiaries
