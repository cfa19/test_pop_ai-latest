# Enroll in a Program

Individual users and beneficiaries can browse and enroll in publicly available programs.

---

## Browse public programs

Navigate to **Dashboard > Programs > Public programs** to see all programs marked as public.

Each program card displays:

- Title and description
- Number of activities included
- Total credit cost
- Organization that created it

---

## Self-enrollment

1. Browse the public program catalog.
2. Click on a program to view its details and included activities.
3. Click **Enroll** to join the program.
4. The program appears in your **Programs** list and you can start working on the first
   activity.

Make sure you have enough credits to cover the program's total cost before enrolling.

---

## Track your progress

Once enrolled, the program detail page shows:

- Your overall progress (e.g., 3 of 7 activities completed)
- Status of each activity (Not started, In progress, Completed)
- Score and validation status for completed activities
