# Create a Program

Build a multi-activity learning path for your beneficiaries.

---

## Access the creation form

Navigate to **Dashboard > Programs** and click **Create program**.

---

## Fill in program details

1. Enter a program **Title**.
2. Write a **Description** that explains the program's objectives and structure.
3. Select a **Category** from the dropdown.
4. Add **Activities** to the program by selecting them from the available catalog. Arrange
   them in the desired order.
5. Optionally set a **Price** (if different from the total credit cost of all activities).
6. Choose whether the program should be **Public** (visible for self-enrollment).
7. Click **Save**.

The total credit cost is calculated automatically as the sum of all included activity costs. Setting a separate program price lets you offer discounts or bundles.

---

## Edit a program

1. Navigate to **Dashboard > Programs**.
2. Click the program you want to modify.
3. Click **Edit**.
4. Make your changes and click **Save**.

Changes to a program do not retroactively affect beneficiaries who are already enrolled. They continue with the program as it was when they started.

---

## Program detail view

The program detail page shows:

- Program title, description, and category
- Total number of activities and total credit cost
- List of included activities (with individual costs and duration)
- Enrollment and completion statistics
