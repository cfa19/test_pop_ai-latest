# Manage Assignments

Assignments connect **beneficiaries** to **activities** or **programs**. Consultants and Organization Admins can create and manage assignments.

---

## Access assignments

Navigate to **Dashboard > Management > Assignments**.

The page displays a summary with four statistics:

| Statistic | Description |
|---|---|
| **Total** | All assignments |
| **Completed** | Assignments where the activity/program is finished |
| **In Progress** | Assignments currently being worked on |
| **Not Started** | Assignments not yet begun |

---

## What you see depends on your role

| Role | Visible assignments |
|---|---|
| **Org Admin** | Beneficiary-to-consultant mappings within the organization |
| **Consultant** | Activity assignments for their own beneficiaries |

---

## Create an assignment

1. Navigate to **Dashboard > Management > Assignments**.
2. Click **New assignment**.
3. Fill in the assignment form:
   - Select a **Beneficiary** (the person who will complete the work)
   - Select an **Activity** or **Program** to assign
   - Optionally set a **Priority** (High, Medium, Low)
   - Optionally set a **Due date**
   - Add **Notes** for the beneficiary
4. Click **Create**.

The form adapts based on your role: - **Org Admin** assigns beneficiaries to consultants within the organization. - **Consultant** assigns activities or programs to their own beneficiaries.

---

## Program assignment detail

For program assignments, a detailed view shows:

- **Beneficiary information** -- Name, email, status, priority, due date
- **Overall progress** -- Progress bar showing completed activities out of total
- **Assignment timeline** -- Enrollment date, last activity date, completion date
- **Validation status** -- Whether the consultant has validated the program
- **Activity list** -- Each activity with its status, score, duration, and credit cost

Overdue assignments display a warning badge next to the due date.

### Validate a program

When all activities in a program are completed:

1. Open the program assignment detail.
2. Review the scores and completion status of each activity.
3. Click **Validate the program**.
