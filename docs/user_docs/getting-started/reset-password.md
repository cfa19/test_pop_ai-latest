# Reset Your Password

If you forget your password, you can request a secure reset link by email.

---

## Request a reset link

1. On the login page, click **Forgot password?**.
2. Enter the **Email** address associated with your account.
3. Click **Send reset link**.
4. A confirmation screen appears. Check your inbox (and spam folder) for the reset email.

You can only request one reset email per 60 seconds. A countdown timer shows the remaining wait time.

The reset link is valid for **1 hour**. After that, you must request a new one.

---

## Set a new password

1. Open the reset email and click the link.
2. On the reset page, enter your **New password**.
3. Confirm it in the **Confirm password** field.
4. Click **Reset password**.

The new password must meet the following requirements:

- Minimum **8 characters**
- At least **one uppercase letter**
- At least **one lowercase letter**
- At least **one digit**

After a successful reset, you are automatically redirected to the login page within 3 seconds.

If the link has expired or is invalid, you will see an error message. Click **Request a new link** to start the process again.
