# First Steps

After logging in, you land on your **Dashboard** — the central hub of POPskills. The dashboard
adapts to your role, showing the most relevant actions and information.

---

## The dashboard at a glance

Depending on your role, the dashboard displays:

- **Statistics cards** -- Key metrics such as active users, completed activities, and credit balance.
- **Onboarding banner** -- Shown during your first 7 days. Provides quick-start buttons tailored to your role.
- **Quick actions** -- Shortcut tiles to the most common tasks.

---

## Quick actions by role

### Individual User

| Action | Description |
|---|---|
| Explore activities | Browse the activity catalog |
| My credits | View your credit balance |
| Purchase credits | Buy credit packs |
| My profile | Edit your personal information |

### Consultant

| Action | Description |
|---|---|
| My beneficiaries | View and manage assigned beneficiaries |
| Assignments | Create and track activity assignments |
| Validations | Review and validate completed activities |
| New assignment | Assign an activity or program to a beneficiary |

### Organization Admin

| Action | Description |
|---|---|
| Manage users | Add and manage team members |
| Assignments | Track activity assignments |
| Purchase credits | Buy credits for the organization |
| Analytics | View organization-wide reports |


---

## Sidebar navigation

The sidebar menu is organized into sections. Items appear or hide based on your role:

1. **Main** -- Dashboard, Activities, Programs, My Results (beneficiary only)
2. **Management** -- Assignments, Beneficiaries, Results, Progressions, Tokens
3. **Administration** -- Users, Organizations, Categories, Audit Logs
4. **Finance & Analytics** -- Credits, Credit Packs, Analytics
5. **Settings** -- Account preferences

If a menu item is not visible, it means your current role does not have access to that feature.
