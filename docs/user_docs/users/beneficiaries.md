# Beneficiary Management

Consultants manage their assigned beneficiaries from a dedicated interface.

---

## Access the beneficiary list

Navigate to **Dashboard > Management > My beneficiaries**.

This page is only available to users with the **Consultant** role.

---

## Beneficiary list

The list shows all users assigned to you as a consultant:

- **Name** and **email**
- **Organization** (if applicable)
- **Number of assigned activities/programs**
- **Overall progress**

---

## View a beneficiary profile

Click on a beneficiary to see their detailed profile:

- **Personal information** -- Name, email, role, organization
- **Assigned activities** -- With status and progress for each
- **Completed activities** -- With scores and validation status
- **Category breakdown** -- Activities grouped by category

---

## Key actions

From the beneficiary profile, you can:

| Action | Description |
|---|---|
| **Create assignment** | Assign a new activity or program |
| **View results** | See completed activity results |
| **Monitor progress** | Check real-time progression on in-progress activities |
| **Validate** | Review and validate completed work |

---

## Monitor progressions

Navigate to **Dashboard > Management > Progressions** to see all your beneficiaries'
in-progress activities in real time.

Each entry shows:

- Beneficiary name and email
- Activity title and type
- Progress bar (percentage)
- Last update time (e.g., "Just now", "5 min ago", "2 hours ago")

Clicking a progression opens the activity in **read-only consultant view**.
