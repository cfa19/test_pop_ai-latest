# Roles & Permissions

POPskills uses a role-based access control system with four user roles. Each role determines
what features, pages, and actions are available.

---

## The four roles

### Organization Admin (Org Admin)

The administrator of a single organization.

- Manage users within the organization
- Create and assign activities and programs
- Purchase credits for the organization
- View organization-level analytics and audit logs

### Consultant

A professional who guides and validates beneficiary work.

- Manage assigned beneficiaries
- Create activities and programs
- Create and track assignments
- Review, enrich, validate, and release results
- Monitor beneficiary progressions in real time

### Beneficiary

An end user assigned to a consultant or organization.

- Complete assigned activities and programs
- View released results with recommendations
- Track personal progress and achievements

### Individual User

A self-service user with no organizational affiliation.

- Browse and run public activities
- Enroll in public programs
- Purchase and manage personal credits
- View activity results immediately (no validation step)

---

## How roles are assigned

- **Individual**, **Consultant**, and **Organization** roles are chosen during
  registration.
- **Beneficiary** accounts are created by consultants or org admins when setting up
  assignments.
