# Audit Logs

Audit logs provide a detailed record of actions performed on the platform, supporting
compliance and troubleshooting.

---

## Access audit logs

Navigate to **Dashboard > Administration > Audit Logs**.

This page is available to the **Org Admin** role.

---

The audit log displays actions within your organization.

---

## Log entries

Each audit log entry records:

- **Action** performed (e.g., user created, activity completed, credits consumed)
- **User** who performed the action
- **Timestamp**
- **Details** (additional context about the action)
